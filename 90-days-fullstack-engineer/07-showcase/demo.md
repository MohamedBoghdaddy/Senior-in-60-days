# Demo: Production-Grade Multi-Service Platform

## What the system does

The platform provides an authenticated search and listing experience, enriched by AI-driven insights.
It combines relational business data with unstructured AI analysis and background processing.

## Key features

- JWT authentication with refresh tokens.
- RBAC-based route protections.
- Search and detail workflows with caching.
- AI insights service with retry and fallback support.
- Docker and Kubernetes deployment scaffolding.

## How to run locally

1. Install backend dependencies and start the service.
2. Install AI service dependencies and start `uvicorn`.
3. Install frontend dependencies and run the React shell.
4. Alternatively use `03-fullstack-system/infra/docker-compose.yml`.
