# .NET Core Clean Architecture Sample

This folder contains a minimal .NET Web API example that demonstrates:
- dependency injection
- middleware
- clean architecture separation
- validation patterns
- logging and error handling

The sample uses:
- a request processing middleware pipeline
- a typed service layer
- API contract models
- centralized exception handling

## Run the sample
1. Navigate to `dotnet`
2. Build with `dotnet build`
3. Run with `dotnet run`

The sample is designed to show structure and senior thought around maintainable C# APIs.
