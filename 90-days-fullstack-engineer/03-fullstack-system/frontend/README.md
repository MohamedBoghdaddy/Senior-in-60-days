# Frontend Shell

This React shell is a lightweight UI that demonstrates a realistic dashboard and AI insight flow.

## Pages included

- `Dashboard` - high-level metrics and quick actions.
- `Listings` - search and pagination for records.
- `Details` - item-level detail with contextual data.
- `AI Insights` - a data-driven score / analysis page.

## Structure

- `src/pages` contains page components.
- `src/components` contains reusable UI pieces.
- `src/services/api.js` centralizes backend communication.
- `src/state/useAppState.js` demonstrates simple client state.

## Run locally

1. Install dependencies: `npm install`
2. Start dev server: `npm start`
