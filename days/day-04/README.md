### Focus

Scaffold the Next.js application with basic routing and layout.
### Timebox

~4 hours
### Study

### Build

1. Initialize a Next.js application using the App Router (`app/` directory).
1. Create a basic layout with header and navigation placeholders.
1. Set up initial pages for sign-in, sign-up, and dashboard.
1. Integrate Tailwind CSS for styling with a configuration file.
